    Frequently Asked Questions

    What is a document converter?

    A document converter changes a file from one format to another, for example turning a PDF into a Word document or an image into a PDF, while keeping the content readable in the new format.

    How do I convert documents online?

    Upload your file, choose the output format, adjust any options like quality or page size, then click Convert and download the result.

    Is this converter free?

    Yes. Every conversion on this page is free to use with no account or sign-up required.

    Are my files secure?

    Most conversions run entirely in your browser and your files never leave your device. A small number of advanced conversions use a secure cloud service, and those files are deleted automatically after processing.

    Does it work on mobile?

    Yes, the converter is fully responsive and works on phones, tablets, and desktop browsers.

    Which formats are supported?

    PDF, DOCX, TXT, HTML, JPG, PNG, WEBP, CSV, XLSX, PPTX, ODT, and RTF, with more combinations being added regularly.

    Do uploaded files remain private?

    Files processed in the browser are never uploaded anywhere. Files that require cloud-based conversion are transmitted securely and removed automatically afterward.

    Can I convert files without registration?

    Yes, no registration or account is required for any conversion.

    Are my files deleted automatically?

    Files used for cloud-based conversions are automatically deleted from the server shortly after the conversion completes.

    Does it work offline?

    Once the page has loaded, browser-based conversions like image-to-PDF and PDF-to-image continue to work offline. Cloud-based conversions require an internet connection.

    Can I convert multiple files?

    Yes, add multiple files to the queue and convert them in a single batch, then download individually or as a ZIP.

    Does formatting stay the same?

    Simple formatting like text, headings, and images is generally preserved. Very complex layouts may shift slightly depending on the conversion type.

    Is there a file size limit?

    Browser-based conversions depend on your device's available memory. Cloud-based conversions have a per-file size limit shown at upload time.

    Does it support mobile devices?

    Yes, the tool is built mobile-first and supports touch-based drag and drop and file selection.

    Can I convert scanned PDFs?

    Scanned PDFs can be converted to images directly. Converting scanned PDFs into editable text requires OCR, which is not yet supported for every format.

    Can I convert password-protected PDFs?

    Yes, enter the password when prompted and the file will be unlocked locally in your browser before conversion.

    Which browsers are supported?

    The latest versions of Chrome, Edge, Firefox, and Safari on desktop and mobile are all supported.

    Can I convert images into PDFs?

    Yes, JPG, PNG, and WEBP images can be combined into a single PDF or converted individually.

    Is my data encrypted?

    Any file sent to the cloud-based conversion service is transmitted over an encrypted connection.

    Can I use it on Mac and Windows?

    Yes, the converter works the same way on Mac, Windows, Linux, Chromebooks, and mobile operating systems, since it runs in the browser.

    Does it support batch conversion?

    Yes, add several files to the queue and convert them together, then download everything as a single ZIP file.
