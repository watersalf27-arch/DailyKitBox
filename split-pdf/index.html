      Frequently Asked Questions

      What is Split PDF?

      Split PDF is a tool that divides one PDF document into multiple smaller PDF files, either by separating every page, extracting specific pages, or dividing the file by custom page ranges.

      How do I split a PDF?

      Upload your PDF, choose a split mode such as custom ranges or individual pages, then click Split PDF. Your files are ready to download in seconds.

      Can I split PDF for free?

      Yes, this tool is completely free with no sign-up, no watermark and no page limit.

      Can I extract pages?

      Yes. Select the exact pages you want using the checkboxes on each thumbnail, then choose the Extract Selected Pages mode to combine them into a single new PDF.

      Does it keep quality?

      Yes. Splitting preserves the original resolution, text, fonts, images and links exactly as they appeared in the source PDF.

      Is my PDF secure?

      Yes. Your PDF is never uploaded to a server. All splitting happens locally in your browser using JavaScript, so your file never leaves your device.

      Can I split large PDFs?

      Yes, there is no artificial page limit. The practical limit depends on your device's available memory for very large files.

      Does it work on mobile?

      Yes, the tool is fully responsive and works on Android and iPhone browsers as well as desktop and tablet.

      Can I split password protected PDFs?

      Password-protected or encrypted PDFs cannot currently be processed. Please remove the password before uploading.

      How do I split a PDF into custom page ranges?

      Choose Custom Ranges mode and type ranges separated by commas, such as 1-3, 5-8, 10-15. Each range becomes its own PDF file.

      Can I download all split files at once?

      Yes. After splitting, click Download All as ZIP to get every resulting PDF in a single compressed file.

      Can I rotate pages before splitting?

      Yes. Each page thumbnail has a rotate button that lets you rotate it by 90 degrees before you split or extract.

      Can I delete pages before splitting?

      Yes. Click the delete icon on any page thumbnail to remove it from the document before splitting.

      Can I reorder pages before splitting?

      Yes. Drag and drop page thumbnails into any order, or use keyboard shortcuts, before running the split.

      Does this tool need an internet connection?

      After the page loads once, it can be installed as an app and used offline since splitting happens locally without a server.

      What file formats are supported for upload?

      Only PDF files are supported for splitting. If you need to convert images or Word documents to PDF first, use the Images to PDF or Office Converter tools.

      Is there a watermark on the split files?

      No. Every file produced by this tool is completely free of watermarks.

      What browsers are supported?

      This tool works on the latest versions of Chrome, Edge, Firefox, Safari and Opera on desktop, Android and iOS.

      Can I undo a mistake while editing pages?

      Yes. Use the Undo and Redo buttons, or the Ctrl+Z and Ctrl+Y keyboard shortcuts, to step back through your recent page edits.

      Can I rename the split files before downloading?

      Yes. Each generated file has an editable filename field so you can rename it before downloading.
